# Commands

1. Setup the environment

    ``` shell
    virtualenv -p python3 venv

    source venv/bin/activate
    ```

2. Install the dependencies

    ``` shell
    pip install -r ./requirements.txt
    ```

3. Create **mkdocs** new project

    ``` shell
    mkdocs new my-project
    ```

4. Serve the project

    ``` shell
    cd my-project
    mkdocs serve
    ```
